# samar3366.github.io

Personal portfolio of **Sumanth Reddy Pathi** — Senior Software Engineer & AI Engineer.

A single-page, dark IDE/terminal-themed site built with plain HTML, CSS and vanilla JS
(no build step — deploys straight to GitHub Pages).

## Structure

| File | Purpose |
| --- | --- |
| `index.html` | Page markup & content |
| `styles.css` | Theme, layout & responsive styles |
| `script.js` | Nav, mobile menu & reveal-on-scroll |
| `icon_*.svg` | Social / contact icons |

## Develop locally

Just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8080
# then visit http://localhost:8080
```
